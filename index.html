Web Components

You can think of Web Components as reusable interface widgets that are created using open Web technologies.

They are rendered directly from the browser, and so there is no need for loading external libraries like jQuery.

An existing Web Component can be used without writing code, simply by adding an import statement to an HTML page.

Web Components

  • Custom Elements
  • HTML Templates
  • Shadow DOM
  • HTML Imports
They consist of several technologies four of the main ones are.

Custom Elements

With Custom Elements, web developers can create new HTML tags, beef-up existing HTML tags, or extend the components other developers have authored. This API is the foundation of web components. It brings a web standards-based way to create reusable components using nothing more than vanilla JS/HTML/CSS.

HTML Templates

HTML Templates elements are used to declare fragments of HTML that can be cloned and inserted in the document by script.

Shadow DOM

Is a method of combining multiple DOM trees into one hierarchy and how these trees interact with each other within a document, thus enabling better composition of the DOM.

HTML Imports

HTML Imports is a way to include and reuse HTML documents in other HTML documents.

Polymer

Polymer is a framework officially developed by Google which adds an abstraction layer on top of Web Components in order to achieve cross-browser compatibility.

it also adds some syntactic sugar so developers can take full advantage of web components

It is currently on version 1.7 and it comes with a full toolbox and a growing element catalog.

The next major release 2.0 is still in the works and is coming some time next year.

Polymer 2.0

  • Web components 1.0
  • ES 6
  • Removes custom abstractions
  • Upgrader
  • Hybrid mode
It is going to support web components 1.0,

developers will be able to write vanilla javascript using ES 6 syntax,

it will remove several of polymer abstractions so developers will be able to work closer to the metal and create custom elements that will be indistinguishable from native custom elements and

finally it will make it easier to upgrade existing elements by offering tools like the upgrader and the hybrid mode.

Toolbox

  • Polymer CLI
  • Lighthouse
  • Linter
polymer CLI is a set of tools that handle operations like generating the app skeleton and starting and stopping the web server and preparing the app for production

Lighthouse is a tool for improving the quality of your web apps. You can run it as a Chrome Extension or from the command line. it runs a barrage of tests against the page, and then it generates a report on how well the page did. From here you can use the failing tests as indicators on what you can do to improve your app.

and a Linter that checks for spelling errors or malformed syntax

PRPL pattern

  • Push critical resources for the initial route.
  • Render initial route asap.
  • Pre-cache remaining components.
  • Lazy-load and create remaining routes on demand.

P.W.A.

Progressive Web Apps

  • Fast
  • Reliable
  • Engaging
Progressive Web Apps is un umbrella term for several web technologies that can be used to transform traditional websites into modern web apps with features that are usually found in native mobile apps like working offline, handling push notifications, being installable and discoverable and able to break free from the barriers of the browser. These apps aren’t packaged and deployed through app stores but they’re just websites that took all the right vitamins. So they are always up to date and thanks to the ask when you need it permission model of the web the user doesnt have to make a heavyweight decision upfornt and unwillingly agree to surrender their privacy by allowing the app to access the camera, microphone or location without their knowledge.

P.W.A.

Progressive Web Apps - checklist

  • Website is served over HTTPS
  • Website works cross-browser
  • Pages each have a unique URL
  • Pages have responsive layouts that render nicely in all devices
  • The start URL loads while offline
  • Metadata is provided for an "Add to Home screen" icon
